Firebase Setup Guide

NPTEL Mock Exam Portal

🏠 Home ⚙️ Admin Panel

☁️ Connect to Firebase

Follow these 5 simple steps to make exams visible to everyone — no server or backend coding needed.

1

Create a Firebase Project

Free — takes about 2 minutes

Go to console.firebase.google.com and sign in with your Google account.

  • Click "Add project"
  • Enter a project name (e.g., nptel-quiz)
  • Skip Google Analytics (optional)
  • Click "Create project" and wait for it to finish
2

Enable Realtime Database

This is where your exam data will be stored

  • In the left sidebar, click Build → Realtime Database
  • Click "Create Database"
  • Choose any location (Singapore is closest for India)
  • Select "Start in test mode" and click Enable
ℹ️
"Test mode" allows anyone to read/write for 30 days. After that you'll need to update the rules (Step 3 below fixes this permanently).
3

Set Database Rules

Allow everyone to read exams, only you to write

In your Realtime Database page, click the Rules tab and replace everything with:

{ "rules": { ".read": true, ".write": true } }

Click Publish. This allows all students to read exams. (For production, you can restrict write access later.)

4

Get Your Firebase Config Keys

These connect your website to your database

  • Go to ⚙️ Project Settings (gear icon at the top-left)
  • Scroll down to "Your apps"
  • Click "Add app" → Web (</>)
  • Give it a nickname (e.g., "nptel-web"), click Register app
  • Copy the firebaseConfig object shown on screen

It will look like this:

// Your config will look like this: const firebaseConfig = { apiKey: "AIzaSyD...", authDomain: "my-project.firebaseapp.com", databaseURL: "https://my-project-default-rtdb.firebaseio.com", projectId: "my-project", storageBucket: "my-project.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" };
5

Paste Config into js/config.js

Open this file and replace the placeholder values

Open the file js/config.js in your project folder and replace each placeholder value with the values from Step 4:

// js/config.js — paste YOUR values here const FIREBASE_CONFIG = { apiKey: "paste your apiKey here", authDomain: "paste your authDomain here", databaseURL: "paste your databaseURL here", projectId: "paste your projectId here", storageBucket: "paste your storageBucket here", messagingSenderId: "paste your messagingSenderId here", appId: "paste your appId here" };

Save the file, then push to GitHub:

git add js/config.js git commit -m "Add Firebase configuration" git push
✅
After pushing, your GitHub Pages site will automatically use Firebase. Any exam you create in the Admin Panel will be visible to all students!

🎉 You're all set!

Once Firebase is connected, exams created in the Admin Panel will be visible to every student who visits your site link.
